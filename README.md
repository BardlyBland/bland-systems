# Bland Systems

Bland Systems is a personal project space for learning, experimentation, and documentation
across IT infrastructure, cloud computing, security fundamentals, and systems design.

This repository contains personal labs, prototypes, and notes created alongside formal study.
It is not a registered company and does not offer professional services.

## Focus Areas
- Cloud and infrastructure labs (Azure, AWS)
- Security fundamentals and best practices
- Systems design and documentation
- Experimental tools and prototypes
- Game and rules system experiments (early-stage)

## Status
Learning-focused · Experimental · Evolving
